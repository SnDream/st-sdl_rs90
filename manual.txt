== How to use ==
D-Pad Up / Down / Left / Right
    Press Up / Down / Left / Right
R Button
    Press ENTER
B Button
    Press BACKSPACE
A Button
    Press SPACE
Select Button
    Show Keyboard
Start Button
    Press TAB
L + R Button
    Press CTRL+C
L + Start Button
    Exit STerminal+

== Keyboard ==
D-Pad Up / Down / Left / Right
    Move on keyboard
L Button (Hold)
    Hold Shift
R Button
    Press ENTER
B Button
    Press BACKSPACE
A Button (Press)
    Press key on keyboard
A Button (Hold > 0.5s)
    Toggle key (useful for shift/ctrl...)
Select Button
    Hide keyboard
Start Button
    Press TAB
L + R Button
    Press CTRL+C
L + Start Button
    Exit STerminal
Select + D-Pad Up / Down
    Switch Keyboard Location

== Cheeatcheet ==
(tutorial at www.shellscript.sh):
TAB key
    complete path
UP/DOWN keys
    navigate history
pwd
    print current directory
ls
    list files (-l for file size)
cd <d>
    change directory (.. = go up)
cp <f> <d>
    copy files (dest can be dir)
mv <f> <d>
    move files (dest can be dir)
rm <f>
    remove files (use -rf for dir)
htop
    see running processes (q to quit)
less <f>
    see content of text file
grep <pattern> <f>
    find in files
